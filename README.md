# HR Management System (HRMS)

A comprehensive Full-Stack HR Management System built with a modern tech stack.

## Tech Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend**: Node.js, Express (Architecture planned)
- **Styling**: Custom Design System with Material Symbols

## Features Implemented
- **Interactive Dashboard**: Real-time stats, payroll overview, attendance trends, and pending approvals.
- **Employee Management**: Profile views, edit details modal, and native document upload capabilities.
- **Settings Module**: Roles & Permissions (with slide-out drawer), Security, Integrations (Slack, AD, Zoom), and General settings.
- **User Profile**: Personal information management.
- **Interactive Workflows**: Dropdowns, modals, toasts, and navigation wired up across the application.

## Getting Started

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

### Backend (Coming Soon)
1. `cd backend`
2. `npm install`
3. `npm run dev`
